刚花了1个小时在GitHub上搭建了自己的博客，参考了很多大神的教程。因为也是新手，搭建过程遇到各种坑。所以想记录下自己搭建的过程，也可以给新手小白们当个参考。如果有错误的地方，欢迎指出！

Hexo + Github 搭建的个人博客： http://jaybin.top/

什么是 Hexo

Hexo 是一个快速、简洁且高效的博客框架。Hexo 使用 Markdown（或其他渲染引擎）解析文章，在几秒内，即可利用靓丽的主题生成静态网页。

什么是 GitHub

GitHub 是通过 Git 进行版本控制的软件源代码托管服务，由 GitHub 公司（曾称 Logical Awesome）的开发者 Chris Wanstrath、PJ Hyett 和 Tom Preston-Werner 使用 Ruby on Rails 编写而成。（维基百科）

不会查的同学，看这里。


二、博客搭建的环境配置

现在要下载2个软件：Git和Nodejs。（都是32位的）

安装Git

（1）官网下载：https://git-scm.com/download/win 选择好自己电脑的版本。我的是win7-32位

（2）下载完成后，双击运行Git-2.20.1-32-bit.exe软件安装包

（3）一直点 Next，继续安装




（4）点击 Install

（5）去掉打钩，再点击finish 完成安装。

（6）键盘按ctrl + r打开 命令窗口, 输入 cmd ，点击确定 。

（7）输入git --version，结果显示 Git 版本，安装成功。（这个命令窗口不用关掉，待会还要用）


或者在桌面单击鼠标右键 看看是否有 Git Bash Here ，都是表示安装成功。

安装Node.js

（1）官网下载：https://nodejs.org/en/download/

（2）下载完成后，双击运行 node-v8.11.3-x86.msi 软件安装包

耐心等待出现 Next 按钮，然后点击

（3）打钩，接受，再点击 Next ,

（4）一直Next

（5）点击 install

（6） 点击 Finsh ，完成安装。

（7）打开刚才的命令窗口，输入 node -v , 然后按 回车 。 再输入 npm -v ,再按 回车

可以看到版本信息，就表示安装成功了。

至此，软件安装步骤完成。

三、搭建 Hexo 博客框架

（1） 安装 Hexo，

在电脑中新建一个文件夹存放自己的博客,名字自己起。例如：D盘里面新建一个 blog 的文件夹
在文件夹内,鼠标右键点击，选择 Git Bash Here 。出现命令窗口.

（2）输入命令：npm install -g hexo-cli ，按 回车 。

（3）初始化 Hexo

在 $ 后面输入 hexo init hexo

完成后得到 hexo 文件夹，里面存放 Hexo 博客所有的文件，包括下面会讲到的主题文件。

（4）在Git Bash 这个窗口继续执行以下代码：（无特别提示，以下代码基本都在 Git Bash 命令窗口执行）

4.1 输入 cd hexo 再按 回车
4.2 输入 npm install 再按 回车
4.3 输入 hexo generate 再按 回车

4.4 启动服务器：执行以下代码 ,输入 hexo server 再按 回车 可以看到服务器代码是 4000

4.5 打开浏览器，地址栏输入http://localhost:4000/ ，结果如下图，可以看到，初始化的 Hexo 博客搭建成功，可以访问。

你可以看到第一篇默认的博客：Hello World。但看起来很难看，后续会通过重新选择模板来对博客进行美化.
好了，现在我们先停止服务器，按Ctrl + C 。 【不要关闭这个窗口，后面还要用】

把你的博客部署到Github Pages上去

1、Github账号注册及配置

  • 注册github帐号
  • 建立new repository

如果你没有github帐号，请到官网注册一个，然后去邮箱进行验证；如果你有帐号则直接登录。
官网：https://github.com/ 点击 sign up .

注册完后，登录进去。

2、新建一个仓库，仓库名为：jaybinkoo.github.io ，如下图所示

3、只需填写 username.github.io 即可，然后点击 create repositrory。 注意：username.github.io 的 username 要和 用户名 保持一致，不然后面会失败。以我的为例：

4、点击github主页右上角的 your profile

5、找到新建立的 username.github.io 文件,点击打开，

6、点击 settings，

7、往下拉动鼠标到GitHub Pages。


如果你看到上方出现以下警告：

不用管他，点击选择 choose a theme，随便选择一个，（之后我们要更改这些丑陋的模板），然后 select theme 保存就行了.

8、将Hexo与Github page联系起来，设置Git的user name和email（如果是第一次的话）

配置 Git 个人信息：在 D:\blog\hexo 目录下，鼠标右键打开一个 Git Bash 窗口，输入下面的命令
8.1 git config --global user.name " GitHub 用户名 " 然后 回车 。【GitHub 用户名】：就是刚才你注册的用户名 例如我的
8.2 git config –global user.email “ GitHub 邮箱 “ 然后 回车。【GitHub 邮箱】：就是刚才你注册的邮箱 例如我的

9、 生成 SSH KEY，其实就是生成一个公钥和密钥，因为 GitHub 需要一个密钥才能与本地相连接。执行以下命令，并 连续按 3 次回车

生成密钥：|输入： ssh-keygen -t rsa -C " 邮箱地址 " 例如我的：ssh-keygen -t rsa -C " jaybin@aliyun.com "

10、 SSH KEY 生成之后,得到了两个文件：id_rsa 和 id_rsa.pub（默认存储路径是：C:\Users\Administrator.ssh）。打开我的电脑，在地址栏输入 C:\Users\Administrator\.ssh ,然后回车，即可打开这个目录 。

11、鼠标右键选择用记事本打开 id_rsa.pub 文件，复制全部内容，即复制密钥。

12、 打开 GitHub ，依次点击 头像 –>Settings–>SSH and GPG keys –> Add SSH key，将复制的密钥粘贴到 key 输入框，最后点击 Add Key ，SSH KEY 配置成功，如下图所示：


13、修改 hexo 文件夹下的 _config.yml 全局配置文件，修改 deploy 属性代码，将本地 hexo 项目托管到 GitHub 上，如下图所示： 鼠标右键选择 记事本打开 ， 修改完后， 一定要另存为，编码选择 UTF-8 ,否则会出现乱码。


14、新建一篇博客，行命令窗口输入：hexo new post “博客名” ，然后 回车. 例如 hexo new post “你好，Hexo”

15、 在这个路径下 D:\blog\hexo\source\_posts 找到刚刚生产的 你好，Hexo.md 文件. 鼠标右键，选择 用记事本 打开。输入 使用Hexo， 是非常简单的一件事 然后保存文件。点击 文件 > 另存为 > 所有文件 > UTF-8 > 保存 > 是

16、在生成以及部署文章之前，需要安装一个 hexo-deployer-git 插件扩展：

输入 npm install hexo-deployer-git --save，然后 回车 。

17、执行下面的代码命令： hexo d -g ，将 hexo 项目以及文章托管到 GitHub 上。

注：hexo generate 可缩写为 hexo g ， hexo deploy 可缩写为 hexo d

18、部署成功后,打开浏览器，输入访问你的地址：http://用户名.github.io。那么将看到生成的文章

[http://jaybinkoo.github.io](http://jaybinkoo.github.io)

好了，到此为止，最基本的也是最全面的hexo+github搭建博客完成。

域名绑定

我的域名是：https://jaybin.top/ .怎么来的呢？
我是看了这篇文章- 4块钱,用Github+Hexo搭建你的个人博客才去绑定 jaybinkoo.github.io 这个域名的。对，没错，就是4块钱。

1、域名获取

我是在 阿里云官网 购买的。
要买域名，首先注册一个阿里云账号。

注册完后，在搜索栏随意输入你喜欢的名字，然后搜索。如果你运气好，域名没有被别人注册，你就选那个吧。
如果被注册了，只能再改动你的名字了。不同的后缀价格不同。.com、.net 、.xin 会比较贵，便宜的后缀是：.top 、.xyz
我那时候买确实是 4元 （搞活动的时候），现在是 9元 了（其实也不贵）。 我买了 3年/67元。平均下来就是20多。后面再续费。

2、域名实名认证

在天朝很多都要实名认证的，既然要你这样做，你就做吧。 当然有能力，你也可以到国外购买就不用认证了。
认证教程点击这里 或者百度一下 个人域名实名认证教程。

3、域名解析


一般填写后，10分钟就可以解析完成了。

4、配置CNAME文件

在博客根目录文件夹下,例如在我的D:\blog\hexo\source 下，新建名为CNAME的记事本文件，然后去掉后缀 .txt。
鼠标右键，用记事本打开。在里面输入你的域名，例如我的：jaybin.top ，/然后，点击 文件 > 另存为 > 选择所有文件 > 选择UTF-8 > 保存。

注意：
这里填不填写www前缀都是可以的，区别在于填写www，那么博客网址就会以www开头，例如：我的博客网址是：www.jaybin.top，二者都可以，看你喜欢。

5、部署你的CNAME文件到GitHub上面。

在命令窗口输入代码： hexo d -g ，再按 回车. （如果它弹出窗口需要你填写 GitHub账户 和 密码，那你就填写，如果没有就不用。）

6、打开你的博客

在浏览器输入你在记事本里的域名网址，即可打开你的博客。
至此，GitHub 绑定域名完成了。赶快把你的域名告诉你的朋友吧！
